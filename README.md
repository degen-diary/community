# Degen Diary — Community Feedback

This is where you shape **Degen Diary**, the psychology-first trading journal for crypto traders.

Report bugs, request features, and discuss ideas. Your feedback directly influences what we build next.

---

## How to participate

| I want to... | Go to |

| Report a bug | [New Bug Report](https://github.com/degen-diary/feedback/issues/new?template=bug_report.yml) |

## Before posting

- **Search first** — Your issue may already exist. Upvote it with a thumbs up instead of creating a duplicate.
- **One issue per report** — Keep each issue focused on a single bug or request.
- **Be specific** — Include steps to reproduce, screenshots, and your browser/device info.
