# Security Policy

## Reporting a vulnerability

**DO NOT** create a public GitHub issue for security vulnerabilities.

Email: **hello@degendiary.app** (subject line: "Security Report")

We will:

- Acknowledge receipt within **48 hours**
- Provide an initial assessment within **5 business days**
- Coordinate disclosure timing with the reporter

## Scope

- degendiary.app (web application)
- API endpoints
- Authentication flows

## Out of scope

- Social engineering attacks
- Denial of service attacks
- Issues in third-party services (Supabase, Stripe, DexScreener)
- Vulnerabilities in software we don't control

## Disclosure policy

We follow responsible disclosure. We will work with reporters to determine an appropriate timeline for public disclosure after a fix is deployed.
